# AI Service Report Assistant

A learning project for building a simple Python assistant for service engineers.

## Current features

- Basic Python scripts
- Service case formatting
- Fault priority detection
- Fault counter
- Service report generation
- Save report to `.txt` file

## Project goal

The goal of this project is to turn rough technical service notes into a structured service report.
